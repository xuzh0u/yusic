# 🎵 音乐调试指南

## 当前版本说明

我已经简化了音乐生成逻辑，使其更加和谐悦耳：

### 改进点

1. **简化的节奏**
   - 钢琴：每小节第1和第3拍
   - 旋律：每拍一个音符，按和弦音顺序
   - 贝斯：每拍一个根音
   - 鼓：标准4/4拍

2. **更柔和的音色**
   - 所有乐器使用正弦波或三角波
   - 降低音量避免刺耳
   - 更短的延音时间

3. **固定的和弦进行**
   - 使用C调的经典进行
   - 避免复杂的转调逻辑

## 如果还是不好听

### 快速调整建议

1. **降低某个乐器音量**
   编辑 `src/music/JazzGenerator.ts`，找到对应乐器的 `volume.value`，
   数字越小越安静（如 -20 比 -10 更安静）

2. **关闭某个乐器**
   注释掉对应的 `this.parts[name]!.start(0)` 行

3. **简化旋律**
   编辑 `src/music/JazzTheory.ts` 的 `generateMelody` 方法

4. **调整 BPM**
   尝试更慢的速度，如 70-80

## 测试建议

1. 先用 Lo-fi Jazz 风格测试
2. BPM 设置为 80
3. 调性选择 C
4. 如果某个乐器太吵，可以先关闭它
